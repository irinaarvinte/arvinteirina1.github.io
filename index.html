Prima mea postare

Bine ai venit pe blogul meu! Aici voi împărtăși din gândurile mele, proiectele creative și lucrurile care mă inspiră zi de zi.

Ce trebuie să faci pas cu pas (totul e gratuit)

  1. Intră pe github.com și creează un cont dacă nu ai deja.
  2. Apasă pe acest link și creează un repository nou numit arvinteirina1.github.io
  3. Bifează Public și Add a README file, apoi click pe Create repository
  4. În pagina nouă, apasă Add file → Create new file, scrie index.html
  5. Copie codul HTML din fișierul generat (acesta) și lipește-l acolo
  6. Jos, apasă Commit new file
  7. Intră în tab-ul Settings al repository-ului → mergi la Pages
  8. La Source alege main și root, apoi apasă Save
  9. Așteaptă 1-2 minute, apoi site-ul tău va fi disponibil la adresa:
    https://arvinteirina1.github.io
